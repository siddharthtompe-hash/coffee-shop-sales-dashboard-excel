 ☕ Coffee Shop Sales Dashboard (Advanced Excel)

 📌 Project Overview
This project is an interactive **Coffee Shop Sales Dashboard** built using **Advanced Excel**.  
It provides insights into sales performance, customer behavior, and product trends.

---

📊 Key Features

✔ KPI Metrics:
- Total Sales: $698,812
- Total Transactions: 149,116
- Avg Bill Per Person: 4.69
- Avg Orders Per Person: 1.44

✔ Interactive Filters:
- Month-wise filtering
- Day-wise filtering

✔ Visual Insights:
- Footfall vs Sales by Store Location
- Category-wise Sales Distribution
- Orders by Weekdays
- Hourly Order Trends
- Top 5 Products by Sales
- Size-wise Order Distribution

---

## 🛠 Tools & Techniques Used

- Microsoft Excel (Advanced)
- Pivot Tables
- Pivot Charts
- Slicers (Month & Day Filters)
- Data Cleaning
- Dashboard Design


📈 Business Insights

- Peak sales occur during morning hours (8 AM – 10 AM)
- Coffee category contributes the highest sales (~39%)
- Weekdays have higher order volume than weekends
- Barista Espresso is the top-selling product

---

🎯 Purpose

This dashboard helps in:
- Monitoring sales performance
- Understanding customer behavior
- Improving product strategy
- Making data-driven business decisions
